h1. Notice for developpers

Business queries are not fully refactored for now.
The list below define equality of methods between query code classes. 

By equality between two methods m1 and m2, we mean that :
- m1 and m2 have the same signature
- m1 and m2 AST equal (comments, annotations '@...' and code format are not take into account in this check)
- methods invoked in the code of m1 and m2 equals too (recursive def)

For instance, the statement below means that method x with no parameters in classes A, B and C equal (according to the previous definition) :
* x()

** B
** C
