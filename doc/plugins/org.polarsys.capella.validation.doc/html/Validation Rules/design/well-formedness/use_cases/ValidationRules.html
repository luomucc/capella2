DWF_UC_01 - Capability realization location check in logical, physical or EPBS level
This rule checks that an Capability Realization can only be located in Logical, Physical and EPBS level.


DWF_UC_02 - Abstract Capability Realization Check
This rule checks that OperationalCapability/Capability/CapabilityRealization element is realized by below layer. Example: If Capability resides in SystemAnalysis layer. It should be realized by a CapabilityRealization from Logical layer.
