TJ_LA_01 - Capability realization refinement check 1
This rule checks that a Capability Realization is linked from a source CapabilityRealization or Capability.


TJ_LA_02 - Capability refinement check
This rule checks that a Capability is refined toward a CapabilityRealization in LogicalArchitecture layer.


TJ_LA_03 - System Function Realization
This rule checks that a System Function is realized by at least one Logical Function.


TJ_LA_04 - System Realization
This rule checks the realization consistency between Logical Component and System.


TJ_LA_05 - System Analysis Realization
This rule checks the realization consistency between Logical Architecture and System Analysis.


TJ_LA_06 - System Actor Realized by Logical Actor
This rule ensures that an Actor is realized by at least one Logical Actor.


TJ_LA_07 - Logical Actor to System Actor Realization
This rule ensures that a Logical Actor realizes at least one System Actor.
